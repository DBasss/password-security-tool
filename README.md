# Password Security Tool

A Python-based password strength checker and secure password generator.

## Features
- Password strength analysis
- Secure password generation
- Memorable passphrase creation
- Common password breach detection

## Usage

Check password strength:
```bash
python password_tool.py check "YourPassword123"
```

Generate secure password:
```bash
python password_tool.py generate
```

Generate passphrase:
```bash
python password_tool.py passphrase
```

## Requirements
- Python 3.6+
